# Infraestrutura Distribuída com Docker

Projeto desenvolvido para a disciplina de Computação Distribuída.

## Tecnologias utilizadas

- Docker
- Docker Compose
- Harbor
- Portainer
- Forgejo
- Nextcloud

## Objetivo

Construir uma infraestrutura distribuída baseada em containers para gerenciamento de serviços e imagens.

## Serviços implantados

- Harbor: registro privado de imagens Docker.
- Portainer: gerenciamento de containers.
- Forgejo: hospedagem de repositórios Git.
- Nextcloud: armazenamento e compartilhamento de arquivos.

## Como executar

```bash
docker compose up -d
